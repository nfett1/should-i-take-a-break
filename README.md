# should-i-take-a-break

> library to indicate if you should take a break or not


## Install

```
$ npm install --save should-i-take-a-break
```


## Usage

```js
const shouldITakeABreak = require('should-i-take-a-break');

console.log(shouldITakeABreak);
// 'Yes' or 'No'
```
